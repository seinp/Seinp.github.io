Desarrollo Web

La Castañuela app

En 2018, Esteban Da Rocha desarrolló una emocionante propuesta para una aplicación web destinada a la administración eficiente de un restaurante y bar llamado 'La Castañuela'. Este sistema innovador incluía un sofisticado control de pedidos por mesa, facilitado por códigos QR, para optimizar la experiencia del cliente y minimizar la dependencia de meseros.

Ready comida a domicilio

En 2020, Esteban lideró el desarrollo de Ready, una aplicación web pensada para facilitar el pedido y entrega de comida a domicilio. Esta plataforma, inspirada en los servicios líderes de entrega rápida, estaba especialmente adaptada a la dinámica de la ciudad de Valera, estado Trujillo, Venezuela.

Desarrollo Android Studio

Momi APP / Momi Medic App

En 2022, desarrollé un sistema de aplicaciones Android que interactuaban entre sí, similar a Uber, pero orientado al ámbito médico para ofrecer consultas médicas a domicilio. Estas aplicaciones gestionaban recetas médicas, historiales clínicos e incapacidades médicas, utilizando tecnologías como Firebase de Google para el almacenamiento de datos NoSQL.
